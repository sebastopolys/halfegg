

* Installation Instructions 

Step 1:

    _Unzip project folders/files under public www domain 

Step 2:

    _ run composer install in console from domain folder

Step 3:

    _Edit required constants with your own information on the config.php file 

Step 4:

    _Run script accessing to the domain in the web browser. 

Step 5

    _Once database is installed, Create admin user



CREATE ADMIN USER:

Use phpmyadmin or similar to edit database and create admin user, 
for security reason this is the only way to create admin user.

Step 1 - Create user

    _ Create register in *database_prefix*_users table:
        _id: 1
        _username: *choose_one* 
        _email: optional
        _password: *choose_one*
        _role: null
        _hash: null

Step 2 - Create item

    _ Create admin item in *database_prefix*_items table:
        _id: 1
        _name: admin
        _type: admin
        _content: -1
        _status: -1

Step 3 - Create user/item relation

    _Create user -> item relation on *database_prefix*_user_item_rel table:
        _type: admin
        _user_id: admin - 1
        _item_id: 1 - admin

Step 4: Login admin

    _Reload page and enter admin account with credentials.


Enter to admin dashboard on Proyect URL  /intralog.php page.
    i.e. -- https://yourdomain.com/intralog.php --

It is necesary to edit the URL of the browser manually 

for security reasons there is no existing link that could drive to backend admin dashboard.
